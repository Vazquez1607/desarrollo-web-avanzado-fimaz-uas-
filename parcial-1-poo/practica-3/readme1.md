# Práctica 3 - Sistema de usuarios con validaciones y excepciones

Este sistema aplica herencia y manejo de excepciones en PHP.
La clase Usuario valida el formato del correo y lanza una excepción si es inválido.
La clase Admin extiende Usuario y define el rol "Administrador".
La clase Alumno extiende Usuario, añade la matrícula y define el rol "Alumno".

Evidencia: Se probó la creación de usuarios válidos e inválidos.
El sistema muestra los datos correctos y controla los errores con mensajes claros.
