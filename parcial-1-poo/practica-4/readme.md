# Práctica 4 - Integración POO + Herencia + Validaciones + Excepciones (PHP 8+)

## 🎯 Objetivo
Construir un mini-sistema orientado a objetos en PHP que integre:
- Encapsulamiento
- Herencia
- Polimorfismo básico (`getRol()` en todas las clases hijas)
- Validación de datos
- Manejo de excepciones con `try/catch`
- Salida en HTML mediante una tabla

- PHP 8+
